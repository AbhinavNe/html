What are forms


  • A form is normally used to get user input.

  • A form has a form element and an input tag.

  • You can make the input to show diffrent things like text, radio, checkboxes, submit and button

Example



How to make a form below. ⬇

        <form>
        <input type="text" id="fname" name="fname" value="">
        <input type="text" id="lname" name="lname" value="">
        <form>
